# Skill: 自动记录提示词 (Record Prompts)

本技能定义了在本项目工作区中，AI 代理与用户进行交互时必须遵守的自动化提示词记录规范。

## 🎯 目标 (Goal)
确保用户的每一次输入和指示都被完整、准确地在本地进行归档记录，便于审计、回溯和知识沉淀。

## 🛠️ 工作流指令 (Workflow Instructions)
1. **识别新提示**：在每个对话轮次（Turn）开始或回复前，检查并提取用户最新发送的提示词文本。
2. **本地文件定位**：将该提示词追加记录在本地路径：`./._local/Prompts.txt` 中。
3. **格式化录入**：
   - 维持递增的阿拉伯数字序号（例如 `13.`, `14.`, `15.` 等）。
   - 保持缩进和换行结构，以便阅读。
   - 提示词条目之间留出空行。
4. **排除机制**：请确保 `._local/` 目录始终处于 `.gitignore` 的排除列表中，切勿将 `Prompts.txt` 提交至公网 Git 仓库。
5. **周期执行**：本项工作应在代理回复用户之前自动、静默地完成。

---
*本技能由 Google Antigravity 引擎自动索引并在运行时全局生效。*
