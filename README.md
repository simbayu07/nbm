# Chrome 书签标签管理器 (Chrome Bookmark Tag Manager)

一款基于 **Chrome Extensions Manifest V3** 规范开发的现代化书签管理工具。它不仅兼容 Chrome 原生的书签系统，而且为其扩展了强大的**标签（Tag）**管理能力。界面采用暗色调与磨砂毛玻璃（Glassmorphism）美学设计，交互响应流畅，能够大幅提升日常书签管理和检索的效率。

---

## ✨ 核心特性

1. **零阻碍自动收藏 (Popup)**
   - 打开插件弹窗瞬间，如果当前网页尚未被收藏，扩展会自动调用 Chrome 原生 API 将其**自动收藏**并点亮黄色星标，无需任何多余操作。
   - 打开弹窗后，输入光标会**自动聚焦于标签输入框**，方便您立即键入标签并按回车保存。
   - 点击星星图标可**一键秒速撤销收藏**，已去除任何多余的确认弹窗。

2. **一键静默快捷收藏**
   - 在任意页面按下快捷键，可在后台快速静默将网页收入书签（无需开启任何弹窗）。
   - 具有**智能角标反馈**机制：收藏成功在图标上闪烁绿底 **`✓`** 提示；若已收藏则闪烁紫蓝底 **`已存`** 提示，并在 2 秒后自动隐藏。

3. **智能标签推荐与联想**
   - 弹窗会根据您的历史标签使用频次，在输入框下方智能推荐 5 个最常用的候选标签，点击即可瞬间补充。
   - 在键入标签时支持实时进行拼写匹配和去重推荐。

4. **全屏仪表盘管理后台 (Dashboard Manager)**
   - 宽敞明亮的两栏式管理控制台，支持对书签进行编辑（Title、URL、Tags）和删除。
   - 支持**多标签交集筛选 (Intersection Filter)**：您可以在侧边栏勾选多个标签，右侧将精准检索出同时包含这几个标签的书签集合，打破传统的单一文件夹限制。
   - 提供“未分类书签”专区，方便快速打扫和收纳无标签的散乱书签。

5. **业界领先的备份与数据恢复 (基于 URL 映射)**
   - 支持一键导出包含全部标签映射关系的 JSON 备份文件。
   - 考虑到在多设备同步或重装浏览器后，Chrome 原生书签的唯一 ID 可能会发生改变，我们的恢复算法**以 URL 作为基准进行匹配同步**，即使用户在另一台电脑上重新导入书签，一键恢复后依然能完美还原您的标签库！

## ⌨️ 默认快捷键支持

| 功能说明 | Windows / Linux 快捷键 | macOS 快捷键 | 状态 |
| :--- | :--- | :--- | :--- |
| **打开快捷收藏与标签管理弹窗** | `Ctrl + Shift + Y` | `Command + Shift + Y` | 默认开启 |
| **一键静默收藏当前网页** (带角标反馈) | `Ctrl + Shift + S` | `Command + Shift + S` | 默认开启 |
| **打开全屏书签管理器控制台** | `Ctrl + Shift + O` | `Command + Shift + O` | 默认开启 |

> [!NOTE]
> 浏览器安全机制不允许扩展重写硬编码的 `Command + D`。如果您希望在 Mac 上使用 `Command + Option + B` 打开我们的管理器后台，请在扩展加载后访问 `chrome://extensions/shortcuts` 进行手动绑定覆盖。

## 🛠️ 项目结构

```text
├── manifest.json         # 扩展程序 Manifest V3 配置文件
├── background.js         # 后台服务脚本 (监听书签清理及快捷键逻辑)
├── popup.html            # 快速收藏弹窗页面结构
├── popup.css             # 快速收藏弹窗样式
├── popup.js              # 快速收藏弹窗交互逻辑
├── manager.html          # 控制台管理后台页面结构
├── manager.css           # 控制台管理后台样式表
├── manager.js            # 控制台管理后台数据交互逻辑
└── icons/                # 插件主题图标 (16x16, 48x48, 128x128)
```

## 🧪 安装与开发调试

1. 克隆或下载本项目到本地（例如工作区：`/Users/simba/projs/nbm`）。
2. 在 Chrome 浏览器地址栏输入 `chrome://extensions/` 访问扩展管理页。
3. 开启页面右上角的“**开发者模式**”（Developer mode）开关。
4. 点击左上角的“**加载已解压的扩展程序**”（Load unpacked）按钮。
5. 选中本项目文件夹 `/Users/simba/projs/nbm` 并确认载入。
6. 固定插件至工具栏，即可立即体验！
